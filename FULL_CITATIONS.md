## HOW TO VERIFY THESE CITATIONS

### Option 1: PubMed Search
1. Go to https://pubmed.ncbi.nlm.nih.gov/
2. Search by PubMed ID (e.g., "19755978")
3. Full citation will appear with abstract

### Option 2: DOI Lookup
1. Go to https://doi.org/ + the DOI number
2. Example: https://doi.org/10.1001/archinternmed.2009.263
3. Will redirect to publisher's page

### Option 3: Google Scholar
1. Go to https://scholar.google.com/
2. Paste the full title in quotes
3. Click on citation link for formatted citations

### Option 4: Library Access
If you have university library access:
- Use your library's journal finder
- Search by journal name and citation details
- Most universities have access to Health Affairs, JAMA Network, etc.
